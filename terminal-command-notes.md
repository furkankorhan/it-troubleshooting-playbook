# Terminal Command Notes

Short notes for common commands used during basic IT checks.

## Network Checks

```bash
ping example.com
```

Checks whether a target is reachable and how long the response takes.

```bash
nslookup example.com
```

Checks DNS resolution.

```bash
traceroute example.com
```

Shows the route to a target on macOS/Linux.

```bash
tracert example.com
```

Shows the route to a target on Windows.

## System and File Checks

```bash
pwd
```

Shows the current folder.

```bash
ls
```

Lists files and folders.

```bash
cd folder-name
```

Changes the current folder.

```bash
mkdir notes
```

Creates a new folder.

```bash
cat file.txt
```

Prints the content of a text file.

## Git Basics

```bash
git status
```

Shows changed files and branch status.

```bash
git add README.md
git commit -m "Update README"
git push
```

Stages, commits and uploads a change.
